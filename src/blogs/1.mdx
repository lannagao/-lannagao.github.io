五月中旬了，陰冷的天气迫使我想要丢掉手头的论文来写点东西。用中文。楼外的草坪上开了几朵蒲公英，似乎可以听懂我在说什么。

翻看之前的备忘录时，无意间看到在二月中的时候写了这样一个清单：
<br/>

“买

一次性筷子盘子

猪肉馅

姜

白菜

饮料

水果

醋？”
<br/>

醋的后面还打了一个问号。我觉得蛮有诗意的。最近的一条备忘录还是：
<br/>

“活着真的很没有意思

真的很痛苦

其实痛苦还是反抗

我也知道我依然在反抗

反抗一直存在的

走到哪里都存在的

永远逃不出的

痛苦

这种痛苦除了我自己没有人会懂

走到哪里也不会消解

我已经放弃

放弃生的那种希望”
<br/>

二月计划和朋友包饺子，在异乡庆祝新年，五月想着去死。感觉这就是漂泊的人所想讲述的所有了。
<br/>

但这里可不是说这些的地方。我准备写一些更加现实的，不那么抽象的，当然也和这些息息相关的东西。
<br/>

1902年，杨度踏上了东渡日本的旅途。我不知道他是否像我晕机那样晕船，是否也会呕吐，服药，期待着下船到不知道哪里的地方好好躺下睡一觉。回想起我为了念书只身来到美国已经是接近六年前的事情了。那次坐飞机也还是毫无意外的恶心晕机。似乎我的身体都在抗拒这趟跨越太平洋的旅程，抗拒这巨大轰鸣的机器和无法掌握在自己手里的命运。六年转瞬，无数变故已无法用语言表述。只能把它们挥洒在论文的前言和后语，成为一次次拿起笔，一次次劝服自己的动机。
<br/>

小石川是一个怎样的地方？明治三十五年，那里是否也曾给过杨度和他的湖南同人一丝安全感？是否因为他和同人的一次次对话与交谈，一起创办的杂志和共同的希冀，而给了他们一丝熟悉和安顿的感觉？他们讲日语的时候是否，也会因为磕磕绊绊而觉得羞耻和不甘？这些微小的情绪和感受，是否也融进了他们日后所作出的变革之中？我有很多的问题，这些问题伴随着我的孤单和茫然，漂浮在我看待这个留学群体的眼光里。
<br/>

“你的不得不捨和遺棄都是守真情的堅持

我留守著數不完的夜和載沉載浮的凌遲

誰給你選擇的權利讓你就這樣的離去

誰把我無止境的付出都化成紙上的一個名字

如今 當我寂寞那麼真

我還是得相信 剎那能永恆

再苦的甜蜜和道理
有時候不得不理” （齐豫 《觉（遥寄林觉民》））
<br/>

“意映卿卿如晤：吾今以此书与汝永别矣！吾作此书时，尚是世中一人；汝看此书时，吾已成为阴间一鬼。吾作此书，泪珠和笔墨齐下，不能竟书而欲搁笔，又恐汝不察吾衷，谓吾忍舍汝而死，谓吾不知汝之不欲吾死也，故遂忍悲为汝言之。”(林觉民 《与妻书》)
<br/>

若林觉民不爱陈意映，他又为什么可以写下这样的文字。如果他爱陈意映，又为什么这下这样的文字。
<br/>

还有十天，我就要踏上寻找他们脚步与意识的旅途…虽然这样的旅途已经持续了接近四年。
